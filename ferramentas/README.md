## Ferramentas e Frameworks de Agentes de I.A

# LangFlow

- LangFlow é uma plataforma low-code para criação de agentes. Sua interface visual é intuitiva e permite até não-programadores criarem PoC's de Agentes. Em vez de escrever códigos extensos, o LangFlow funciona conectando componentes para montar fluxos de conversação e pipelines de Agentes. Os casos de uso indicados para o LangFlow são projetos de menor complexidade e prototipagem rápida de Agentes.

# LangChain

- LangChain é um framework para construir agentes em hard-code (via código). Essa biblioteca fornece classes Python para montar pipelines de Agentes envolvendo LLMs. Aqui você escreve código para instanciar modelos, definir prompts e conectar ferramentas, o que oferece maior flexibilidade e controle total sobre cada detalhe da implementação.
